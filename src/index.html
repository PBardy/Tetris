Controls

Move pieces across the grid with either the A D or LEFT and RIGHT arrow keys respectively.

Rotate pieces with either the W or UP arrow keys.

Soft drop, which moves pieces down quicker, but doesn't fully drop them, using the S or DOWN arrow keys, and hard drop using SPACE.

Press ANY key to start.
